  1. Inline element
    • Width and height are ignored. Margin and padding push elements away horizantally but not vrtically.
  2. Block element
    • Block elements break the flow of a documen. Width, Height, Margin and Padding are respeced.
  3. Inline-Block element
    • Behaves like an inline element except Width, Height, Margin and padding are respected.
