CSS Units Explained

Realtive Units

Percentages are always relative to some other value.


Sometimes, its a value from the parent and other times it's a value from the element itself.

Em's are relative Units : With font-size, 1em equals the font-size of the parent. 2em's is twice the font-size of the parent, etc.

With other properties, 1em is equal to the computed font-size of the element itself. When em's are used the elements change relative to their parent. See the example of button on how it changes wrt to parent.

REM

To rectify backdrop of EM, REM(Root EMS) are used. Relative to the root html element's font-size. Often easier to work With. If the root fonst-size is 20px, 1 rem is always 20px, 2 rem is always 40px, etc.
