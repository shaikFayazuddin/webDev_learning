Async Fucntions

  • A newer and cleaner syntax for working with async code!
  • Syntax makeup for promises
  • Async functions always return a promise
  • If the function returns a value, the promise will be resolved without the value
  • If the function throws an exception, the promise will be rejected

Await Keyword

  • We can only use the await keyword inside of functions declared with async
  • await will pause the execution of the fucntion, waiting for a promise to be resolved
