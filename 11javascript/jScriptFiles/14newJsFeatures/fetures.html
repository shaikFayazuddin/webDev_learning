New Java Script Features

  • Spread
    • Spread sysntax allows an iterable such as an array to be expanded in places where zero or more arguments (for fucntin calls) or elements (for array literals) are expected, or an object expressin to be expanded in palces where zero or more key-value pairs(for object literals) are expected.
  • The Arguments Object
    • Available inside every function
    • It's an array-like object.
      • Has a length propery
      • Does not have array methods like push/pop
    • Contains all the arguments passed to the function
    • Not Available inside of arrow fucntions.
  • Rest Params
    • Collects all remainig arguments into an actual array
