Flexbox is a one-dimensional layout method for laying out items in rows or columns. It is used to address common layout frustations. Flexbox allows us to distribute space dynamically across elements of an unknow size, hence the term flex.

Flex-basis: Defines the intial size of an element before additional space is distributed

Flex-Grow: Controls the amount of available space an element should have take up. Accepts a unit-less number value

Flex-shrink: If items are larger than the container, they shrink accoring to flex-shrink.
